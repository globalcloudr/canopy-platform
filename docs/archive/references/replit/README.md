# Replit Reference Projects

These projects are kept here as reference material that may inform future Canopy products, integrations, or plugin work.

They are not currently treated as the active Canopy platform implementation.

## Current References

- `akkedis-digital-platform`
  - reference material for portal or client-platform ideas
- `asap-integration`
  - reference material for `Canopy Web` and ASAP-related integration work
- `issuu-press-connect`
  - reference material for `Publish Canopy`
- `success-story-engine`
  - reference material for `Reach Canopy`
- `sendgrid-newsletter-pro`
  - reference material for `Community Canopy`

## Working Rule

If a reference project graduates into real Canopy product or plugin work, it should be:

- renamed to fit the Canopy product structure
- moved into an intentional repo location such as `apps/`, `plugins/`, or a dedicated product repo
- cleaned of transient Replit-specific files and nested repo history as needed
